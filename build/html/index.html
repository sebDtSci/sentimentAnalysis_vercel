src.app.home()¶

Processes GET and POST requests on the ‘/’ root route.

For a GET request, displays an HTML page without preloaded product data. For a POST request, extracts the text submitted via the form, performs an analysis (for example, calculates a price based on the text) and returns the same HTML page with the product data (the result of the analysis) included.

Returns:

The HTML content of the home page, with or without the product data included, depending on whether the request is GET or POST.

Return type:

str
